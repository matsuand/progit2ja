1.4 はじめよう-コマンドラインツール

コマンドラインツール

Git を利用する方法はいろいろあります。 元々はコマンドラインツールとして提供されていますが、さまざまな機能を持ったグラフィカルユーザーインターフェースもあります。 しかし本書ではコマンドラインツールを用いていくことにします。 Git コマンドをもれなく実行できるのはコマンドラインツールだけです。 GUI はわかりやすさを考慮して、その一部の機能しか実現していないものがほとんどです。 コマンドラインツールを用いて実行する方法がわかっていれば、GUI 版でも実行できるはずです。 ではこの話が逆であっても成り立つかいうと、そうではないでしょう。 また、グラフィカルなツールを選ぶのは好みの問題ではありますが、少なくともコマンドラインツールは、git をインストールしたマシンならどれでも利用できるはずです。

そこでここから先は、macOS でいえばターミナルの開き方、Windows でいえばコマンドプロンプトや Powershell の実行の仕方がわかっているものとして話を進めます。 この内容がわからないなら、いったん読み進めるのはやめにして、よく調べてからにしてください。 それを終えてから、この先の説明や例に進むようにしてください。
