A.4 IntelliJ - PyCharm - WebStorm - PhpStorm - RubyMine における Git

IntelliJ / PyCharm / WebStorm / PhpStorm / RubyMine における Git

JetBrains IDE (たとえば IntelliJ IDEA、PyCharm、WebStorm、PhpStorm、RubyMine など) では、Git 統合プラグインが提供されています。 このプラグインによって IDE 内に専用画面が開いて、Git や GitHub におけるプルリクエスト作業を行うことができます。

図 159. JetBrains IDE 内のバージョン管理ツールウィンドウ

統合プラグインは Git のコマンドラインクライアントを利用しているため、クライアントをインストールしておく必要があります。 公式ドキュメントは https://www.jetbrains.com/help/idea/using-git-integration.html から入手できます。
